## 其他

1. 系统最低要 Android 7.0
2. 第一次开启APP授予权限即可
3. 初始界面是不需要无障碍和悬浮窗的权限，点击开启悬浮窗会
4. 无障碍开启方法:点击悬浮窗启动后转到无障碍设置界面，在里面的应用程序选择这个 APP，开启就OK，不过手机带有悬浮球可能无法开启
5. 结束的标志一般是弹出对话框表示结束，或者就是报了 error 强制停止
6. 关于日志的问题，在脚本 APP 主界面点击返回能看到 log 信息
